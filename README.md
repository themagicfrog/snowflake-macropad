# Snowflake Macropad

This is a 2x2 macropad themed snow for Hack Club's Blueprint/Hackpad/Prototype. I like keyboards a lot and also it's winter which is why I chose to
make it winter-themed with the snowflake silkscreen. It also has 2 LED lights. This was fun to make!

# BOM
- 4x SW_Cherry_MX_1.00u_PCB Switches
- 1x XIAO-RP2040-DIP 
- 2x LED_SMD:LED_SK6812MINI_PLCC4_3.5x3.5mm_P1.75mm
